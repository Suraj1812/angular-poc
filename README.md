# Angular Survey POC

This project is a proof of concept (POC) for a multi-tenant survey management system with role-based access control (RBAC), built using Angular.

The goal of this assignment is to demonstrate:
- Angular architecture
- Multi-tenant routing
- Role-based access control
- Survey management
- Dashboard visualization
- Mock API handling (no backend required)
